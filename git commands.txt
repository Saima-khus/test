there are 2 ways to create and make reposities available on the github:
way 1: if you have already existing code/folder in your computer, then you will have create a repository on github and then connect the local folder with the created repository
Create new repository and connect the repository with an existing folder on your computer

1. Using terminal, from your existing folder, initialize git
git init
2. push this initiated folder (we should call it local repository) to an existing remote repository:
git init -b main
git add --all
git commit -m "My first commit"
git remote add origin git@github.com:saima-khus/Learn-Cpp.git
git push -u origin main


git remote add origin https://github.com/saima-khus/Learn-Cpp.git
git branch -M main
git push -u origin main


way 2: if you have not written any code, then it is easy. First create a repository, then you can clone the repository in your computer
